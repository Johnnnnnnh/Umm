# Tienda Virtual UMM

Proyecto académico de tienda virtual básica.

## Funcionalidades
- Carga productos desde `data/productos.json`.
- Muestra productos en tarjetas con nombre, descripción, precio y stock.
- Carrito de compras dinámico.
- Formulario de compra con validaciones.
- Formulario de contacto.

## Cómo ejecutar
1. Clonar el repositorio.
2. Abrir `index.html` en un navegador.
3. (Opcional) Servir con un servidor local para evitar problemas de `fetch`.
